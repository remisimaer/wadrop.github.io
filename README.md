## Contribute
Open a merge request with your contribution.

If you are not tech-savvy, you can simply open an issue and submit your contribution. We'll handle the technical implementation for you.

**Important:** If you contribute audiovisual content (images, videos, audio files), you must own the rights to that content or have explicit permission to use it. By submitting content, you confirm that you have the necessary rights and permissions.

## Installation

### Installing Ruby & Jekyll

If this is your first time using Jekyll, please follow the [Jekyll docs](https://jekyllrb.com/docs/installation/) and make sure your local environment (including Ruby) is set up correctly.

### Installing the project

Download or clone the repo.

To run the project locally, navigate to the project directory and run:

```
bundle install
```

To start the Jekyll local development server:

```
bundle exec jekyll serve
```

To build the project:

```
bundle exec jekyll build
```
