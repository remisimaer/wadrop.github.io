## 貢献方法

WaDrop.org を「日本の Drupal 情報を最も包括的に集約した場所」にするため、コミュニティからの貢献を歓迎します。

### 開発者の方へ

GitHub リポジトリに Pull Request を作成してください。

[GitHub を見る](https://github.com/remisimaer/wadrop.github.io)

### 技術的な作業が不慣れな方へ

Issue を立てて内容を投稿してください。技術的な反映作業はこちらで行います。

[Issue を作成](https://github.com/remisimaer/wadrop.github.io/issues)

### ディスカッションに参加

質問やアイデアがあれば、GitHub Discussions で気軽に話しましょう。

[Discussions に参加](https://github.com/remisimaer/wadrop.github.io/discussions)

* * *

## 貢献できる内容

- 企業プロフィールの追加／更新
- 日本向け Drupal モジュールの追加
- コミュニティ情報の共有
- ガイドやドキュメントの改善
- リンク切れ／古い情報の報告
- 翻訳
- 新しいカテゴリや機能の提案

**重要：** 画像・動画・音声などの視聴覚コンテンツを提供する場合は、そのコンテンツの権利を保有しているか、使用許諾を得ている必要があります。投稿により、必要な権利・許可を有していることを確認したものとみなします。
