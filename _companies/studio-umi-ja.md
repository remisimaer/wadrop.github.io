## 基本情報

Studio Umi は Drupal を用いたデザイン、開発、運用に強みを持つ日本の企業です。企画、Web マーケティング、インフラ、UI デザイン、開発、運用までを包括的に提供し、複雑でスケーラブルな Web ソリューションの実現を支援します。

## Drupal 関連サービス

- **システム開発**：拡張性を前提とした設計・構築
- **コンサルティング／監査**：要件整理、互換性・性能面のリスク評価、改善提案
- **保守・運用**：継続的な改修と安定運用を支える保守契約
- **他システム連携**：外部基盤との統合による機能拡張
- **アジャイル／ノーコード・ローコード支援**：初期構築後の運用・改善をしやすい設計

## 対応業界

- **教育**：学術アーカイブ系ポータル等
- **大企業**：スケーラブルでセキュアな Web 基盤
- **EC**：商品・受注管理を含む EC サイト

## 主な実績・事例

- **EC の構築・移行**：Drupal を用いた EC サイトの構築・移行と運用改善
- **DX 支援**：DX 推進体制の立ち上げやデジタル施策を支える基盤づくり

## ポジショニング

Studio Umi は、Drupal を中核に据えたエンタープライズ向け Web 基盤の構築・運用に強みを持つ技術パートナーです。アジャイルで柔軟な進め方により、要件変化にも対応しながら価値を積み上げます。

## 認定・パートナーシップ

- ISO/IEC 27001 / JIS Q 27001
- Acquia Certified Site Builder / Developer
- Acquia / Drupal Association との連携、コミュニティ活動
